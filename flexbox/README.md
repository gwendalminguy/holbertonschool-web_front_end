## Flexbox

This project is about learning to use Flexbox.

* [images/](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/flexbox/images), the directory containing image files.

* [*-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/flexbox/10-index.html), the HTML file containing the structure of the web page.

* [*-article.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/flexbox/14-article.html), the HTML file containing the structure of an article web page.

* [*-styles.css](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/flexbox/14-styles.css), the CSS file containing the styles of the web page.
