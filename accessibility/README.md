## Accessibily

This project is about learning to improve Accessibility.

* [images/](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/accessibility/images), the directory containing image files.

* [fix-a11y/](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/accessibility/fix-a11y), the directory containing general accessibility improvements.

* [keyboard/](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/accessibility/keyboard), the directory containing keyboard accessibility improvements.

* [skip-links/](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/accessibility/skip-links), the directory containing links skipping accessibility improvements.

* [00-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/accessibility/00-index.html), the HTML base file containing the structure of the web page.

* [00-article.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/accessibility/00-article.html), the HTML base file containing the structure of an article web page.

* [00-styles.css](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/accessibility/00-styles.css), the CSS base file containing the styles of the web page.
