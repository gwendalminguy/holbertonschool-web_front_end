## Bootstrap

This project is about learning to Bootstrap.

* [0-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/0-index.html), an HTML file with Bootstrap initialized.

* [1-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/1-index.html), an HTML file to practice breakpoints and grid manipulation.

* [2-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/2-index.html), an HTML file to practice table manipulation.

* [3-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/3-index.html), an HTML file to practice cards manipulation.

* [4-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/4-index.html), an HTML file to practice tab and accordion manipulation.

* [5-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/5-index.html), an HTML file to practice breakpoints and display manipulation.

* [6-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/6-index.html), an HTML file with pricing cards.

* [6-styles.css](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/Bootstrap/6-styles.css), a CSS file to override style properties.
