## SASS - SCSS

This project is about learning to use SASS and SCSS.

* [0-debug_log.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/0-debug_log.scss), a SCSS file that prints "Hello world" in the debug output.

* [1-color_variable.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/1-color_variable.scss), a SCSS file that assigns the text color #3D3D3D to the 'body' and 'p' HTML tags.

* [2-color_variables.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/2-color_variables.scss), a SCSS file that assigns the text color #3D3D3D to the 'body' and 'p' HTML tags, and the background color #6D6D6D to the 'body' and 'h2' HTML tags.

* [3-nested_tag.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/3-nested_tag.scss), a SCSS file that assigns no margin or padding in 'body' tags, and a margin of 10px to all of the 'p' tags inside 'body' tags.

* [4-nested_class.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/4-nested_class.scss), a SCSS file that assigns the text color #3D3D3D to elements inside 'body' tags and the text color #FF0000 to any elements of class .red inside 'body' tags.

* [5-nested_child.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/5-nested_child.scss), a SCSS file that assigns the text color #3D3D3D to elements inside 'body' tags and the text color #FF0000 to any elements of class .red that are the first children of the 'body'.

* [6-nested_hover.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/6-nested_hover.scss), a SCSS file that assigns the text color #FF0000 to 'button' tags and changes the color to #00FF00 on hover.

* [7-nested_deeper.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/7-nested_deeper.scss), a SCSS file that assigns the font size 14px to all 'body' tags, the font size 16px to all 'h1' tags inside 'body' tags and the font size 12px to 'h1' tags of class .smaller inside 'body' tags.

* [8-mixin_margins.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/8-mixin_margins.scss), a SCSS file that assigns margin left and right at 10px to 'body' tags and margin left and right at 15px to 'div' tags.

* [9-extend_list.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/9-extend_list.scss), a SCSS file that assigns the font size 12px to all tags of class .info, the text color #00FF00 to all tags of class .success and extend style of the class .info, and the text color #FF0000 to all tags of class .warning and extend style of the class .info.

* [10-colors.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/10-colors.scss), a SCSS file defining three colors.

* [10-import_colors.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/10-import_colors.scss), a SCSS file that assigns the text color $red from '10-colors.scss' to the class .red, the text color $green from '10-colors.scss' to the class .green, and the text color $blue from '10-colors.scss' to the class .blue.

* [11-names.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/10-colors.scss), a SCSS file defining a list of names.

* [11-loop_photos.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/11-loop_photos.scss), a SCSS file that creates a class for each name in the list $list-names from '11-photos.scss' and assigns the background image based on the name.

* [12-loop_header.scss](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/sass_scss/12-loop_header.scss), a SCSS file that creates h* tags, where '*' is the size of the font used (h1: 1px, h2: 2px, etc.) from 'h1' to 'h5'.
