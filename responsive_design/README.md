## Responsive Design

This project is about learning to make designs responsive.

* [images/](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/responsive_design/images), the directory containing image files.

* [*-index.html](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/responsive_design/00-index.html), the HTML file containing the structure of the web page.

* [*-styles.css](https://github.com/gwendalminguy/holbertonschool-web_front_end/tree/main/responsive_design/00-styles.css), the CSS file containing the styles of the web page.
